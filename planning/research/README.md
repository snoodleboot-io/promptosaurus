# Planning Research

This directory contains research and analysis documents used to inform planning decisions.

## Purpose

Capture exploratory research, competitive analysis, technical investigations, and feasibility studies that support planning activities.

## Contents

Typical research documents:
- **Technical Research**: Framework comparisons, library evaluations
- **Competitive Analysis**: How other tools solve similar problems
- **Feasibility Studies**: Can we build this? What are the constraints?
- **Performance Analysis**: Benchmarks and performance investigations
- **User Research**: Understanding user needs and pain points

## File Naming Convention

Use `.research.md` extension to distinguish from planning docs:
```
AI_TOOL_CAPABILITY_MATRIX.research.md
EXECUTION_MODELS_VERIFIED.research.md
PERFORMANCE_BENCHMARKS.research.md
```

## Usage

- Reference research when creating PRDs and ADRs
- Update research as new information becomes available
- Archive outdated research to keep directory relevant

## Relationship to Planning

Research feeds into:
1. **PRDs** - User research informs requirements
2. **ADRs** - Technical research supports architectural decisions
3. **Execution Plans** - Feasibility studies inform task breakdown

## Guidelines

- Keep research objective and data-driven
- Date your research (context degrades over time)
- Link to sources and references
- Update or archive when information becomes stale
