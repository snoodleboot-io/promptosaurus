# User Research Guide

## Goal
Gather deep insights about user needs, behaviors, and pain points to inform product decisions.

## Quick Steps

1. **Define Research Goals**
   - What decisions will this inform?
   - Key hypotheses to test

2. **Choose Methods**
   - Interviews (qualitative depth)
   - Surveys (quantitative scale)
   - Usability testing (behavior observation)

3. **Recruit Participants**
   - Target user segments
   - 5-8 for interviews, 30+ for surveys

4. **Conduct Research**
   - Prepare discussion guide
   - Record sessions, take notes

5. **Analyze Findings**
   - Identify patterns and themes
   - Create user personas

6. **Share Insights**
   - Research report with recommendations
   - Update product requirements

## Key Frameworks

- **Jobs to Be Done:** Focus on user outcomes
- **User Journey Mapping:** Understand full experience
- **Persona Development:** Represent user segments

## Success Factors

- Clear research questions defined
- Representative user sample
- Unbiased data collection
- Actionable insights generated
- Findings influence decisions
